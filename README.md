# StudentPerformancePredictor
This project was done as part of our assessment in CSE422 (Artificial Intelligence) course. It was a collaborative project with my friend Chhoya. In this project we to tried to explore the different factors affecting the academic performance of a student using Machine Learning techniques. Initially we did classification analysis. For this, we have selected a comprehensive dataset from Kaggle to analyze patterns and correlations between the various attributes and the outcomes present in the dataset. Although the dataset contained contained continuous exam scores as target variables, we converted the scores into grades, thus making it possible to do classification analysis since we were having problem running the models for regression analysis. We then trained 3 models: K-Nearest Neighbors (KNN), Logistic Regression and Naive Bayes getting accuracies of 83%, 98% and 92% respectively. Later on, we decided to do regression analysis too, using Linear Regression and KNN algorithms, which was later implemented in this website using flask API and server and models trained with scikit-learn saved as pickle files.
